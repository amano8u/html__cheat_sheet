HTML and CSS References

Tables Tags

Tag Name Description Notes
<table> Table This is used to start the set up a Table in HTML. If you are using a Text editor such as VS Code the closing tag will be triggered immediately.
<thead> Table Header This tag indicates the following columns are headingss in that table.
<th> Table heading data The text between this tag will be the one that appears on the browser
<tbody> Table Body This tag wraps all the content that is considered to be the body data in the table. Is useful for styling the content separately in the table.
<tr> Table Row Each set of the <tr> tag creates a new row for you to enter the data in.
<td> Table Data Each set of the <td> tag contains the data for that particular row and column.

Tables Attributes

Tag Name Description Notes
<colspan> Column Span This is used to start the set up a Table in HTML.
<rowspan> Row Span This tag indicates the following columns are headingss in that table.

Forms Tags

Tag Name Description Notes
<form> Form This set of tags will create a form for users to enter in details in HTML. If you are using a Text editor such as VS Code the closing tag will be triggered immediately.
<input> Input This set of tags creates the various input fields in the HTML form.
<label> Label This set of tag allows you to label the various input fields on the form, to give your users better understanding what is needed.

Forms Atributes

Tag Name Description Notes
<action> Action This attribute defines the action of what the HTML form is meant to do. If you are using a Text editor such as VS Code the closing tag will be triggered immediately.
<target> Input This set of tags creates the various input fields in the HTML form.
<label> Label This set of tag allows you to label the various input fields on the form, to give your users better understanding what is needed.
<method> Method This set of tag defines how the form should be handling the data that has been entered in the form.
